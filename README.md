# LIRI
LIRI - an application to search tweets, songs, and movies

APIs used:
>Twitter
>Spotify
>OMDB


To run this application:

'node liri.js **option** *selection*'

Options are as follows:
- my-tweets
- spotify-this-song
- movie-this
- do-what-it-says

Selections can be any song or movie title you'd like to search
